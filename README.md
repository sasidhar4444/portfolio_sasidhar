## üìù Content Updates

To update the content, edit `index.html`:

- **Resume**: Replace `/mnt/data/Sasidhar-Resume.pdf` with your actual resume file path (e.g., `assets/resume.pdf`).
- **Images**: Replace the SVG placeholders in `index.html` with actual `<img>` tags pointing to your project screenshots or profile picture.
- **Projects**: Edit the `<article>` blocks in the `#projects` section.

## üé® Customization

Edit `styles.css` to change the look and feel:

- **Colors**: Modify the CSS variables in `:root` (e.g., `--accent-primary`).
- **Typography**: Change `--font-sans` to your preferred font family.
